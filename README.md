# scratch-jatek

Ez a Scratch-ban készített játékom.

## Leírás 
A játék célja a "túlélés" a szellemmel szemben. Ha a játékos 3 élete elfogy a játéknak vége.

## Irányítás
-wasd = mozgás 
-szóköz = lövés
